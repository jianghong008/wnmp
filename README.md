# wnmp

#### 介绍
基于c# wpf，wnmp PHP集成管理

[安装本程序](https://gitee.com/jhpw/wnmp/releases/tag/2021-12-01)

#### 功能介绍
1. windows平台php集成开发环境。一键管理。
2. 软件版本任意切换（通过本平台下载或自行从官网下载软件），不用担心不会配置，系统自动配置好。
3. php扩展配置可视化，nginx站点可视化配置。
